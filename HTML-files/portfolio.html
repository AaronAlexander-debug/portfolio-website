Google Homepage Project • completed June 2021

My first fully completed project! All credit goes out to The Odin Project, which has helped to facilitate the development of this project and previous projects during the summer of 2021. There is little to no real functionality on the faux-Google homepage, but it features the search bar, a fully optimized nav and footer bar, and basic HTML and CSS functions such as box shadows, hovering mechanics, and the manipulation of padding and margins to change the positions of the objects on the page.
